
# Fitness Tracker App

## Core Features

### User Profile & Goals
* Store basic user info (name, experience level, goal)
* Goals like "build strength", "build muscle", "improve endurance", "lose fat"

### Exercise Library
* Predefined list of exercises with attributes (muscle group, equipment, difficulty)
* Ability to add simple custom exercises

### Workout Plans
* Create custom workouts by combining exercises with sets/reps/rest
* Auto-generate workouts based on user goals (e.g., strength vs endurance templates)
* Edit, duplicate, and delete workouts

### Workout Execution & Logging
* Start a workout session from a saved routine
* Log what was actually done (sets, reps, weights, perceived difficulty)
* Track date/time and completion status

### Progress & Basic Analytics
* View recent workouts
* See simple stats like total workouts per week, top exercises used
